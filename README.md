# Проект "Лабиринт сокровищ"
### Домашнее задание №1. Текстовая приключенческая игра "Лабиринт сокровищ".
#### Выполнил студент Хилалов Муслим, группа М25-555

## Описание игры

Цель игры - открыть сундук с сокровищем в сокровищнице. Для достижения этой цели вы должны исследовать различные комнаты лабиринта и находить полезные предметы. Иногда при вашем передвижении могут происходить случайные события, которые будут влиять на ваш игровой процесс. В некоторых комнатах содержатся загадки, за правильное решение которых вас может ожидать награда. Также в игре есть комната с ловушкой. Будьте осторожны и крайне внимательны здесь. В общем, вас ждёт интересное приключение. Удачи!

## Установка и запуск игры

|Команда (make)|Команда (poetry)|Описание|
|:-:|:-:|-:|
|make install|poetry install|Установка зависимостей игры|
|make project|poetry run project|Запуск игры|
|make build|poetry build|Сборка пакета|
|make package-install|python3 -m pip install dist/*.whl|Установка пакета|

## Интерфейс

Игровой интерфейс на каждом шаге выглядит следующим образом:

|Команда|Описание|
|:-|--------:|
|go <direction>|перейти в направлении (north/south/east/west)|
|north|shortcut-команда для go north|
|east|shortcut-команда для go east|
|south|shortcut-команда для go south|
|west|shortcut-команда для go west|
|take <item>|поднять предмет|
|use <item>|использовать предмет из инвентаря|
|inventory|показать инвентарь|
|solve|попытаться решить загадку в комнате|
|quit|выйти из игры|
|help|показать это сообщение|
